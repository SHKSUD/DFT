# Images

Place the following files here before deploying:

| File | Size | Purpose |
|---|---|---|
| `og-image.jpg` | 1200×630px | WhatsApp / Twitter / LinkedIn share preview |
| `favicon.ico` | 32×32px | Browser tab icon |
| `favicon-32.png` | 32×32px | High-res favicon |

All three are referenced in `index.html` head and `vercel.json`.
